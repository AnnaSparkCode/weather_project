## Weather & Climate Data Analyzer

This program can read, process, and analyze real-world weather data and produce clear summaries and visualizations.

To use the project, please download the following datasets place the zip files in the "data" folder and extract them there.

- [Weather Data](https://www.kaggle.com/datasets/prasad22/weather-data) from Kaggle.
- [Weather Data](https://data.open-power-system-data.org/weather_data/) from Open Power System Data.
